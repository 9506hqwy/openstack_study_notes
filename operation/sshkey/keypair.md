# keypair

## SSH 鍵の作成

SSH 鍵を作成する。

```sh
ssh-keygen -q -N "" -f demo_rsa
```

## SSH 公開鍵の登録

```sh
openstack keypair create --public-key demo_rsa.pub mykey
```
